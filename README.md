# Two-Choice Quiz App

# 1. Purpose
This project is a simple two-choice quiz application.
It helps users learn through quick decision-based questions.

# 2. Technologies Used
- Python
- Random module

# 3. Features
- Score tracking
- Instant feedback (Correct / Incorrect)

# 4. How It Works
The program displays a question with two options.
The user selects one option.
The program checks the answer and updates the score.
